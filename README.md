# Cardápio

Projeto demo de cardápio digital com frontend estático e backend em API edge (Vercel).

## Como testar localmente
1. Instale Vercel CLI: `npm i -g vercel`
2. Execute: `vercel dev`
3. Acesse: `http://localhost:3000`

## Estrutura
- `src/index.html`
- `src/style.css`
- `src/script.js`
- `api/cardapio.js`
- `vercel.json`
